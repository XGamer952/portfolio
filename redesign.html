Responsive Redesign

Transforming a public website with increased clarity, usability, and modern accessibility.

I chose to redesign the website for Pacific Northwest X-Ray Inc. because it's publicly accessible, not particularly popular, and has a very jarring and inaccessible design.

The original website suffers from severe usability issues: dated visuals, low color contrast, inconsistent layout, and a lack of responsive design.

My redesign focuses on clear visual hierarchy, mobile-first responsiveness, and a smoother shopping experience.

Identifying Issues

  • Poor contrast (e.g., gold text on white background)
  • Text-heavy layout with minimal visual breaks
  • Redundant navigation paths and unclear labels
  • No responsiveness: site breaks entirely on mobile
  • Unclear visual hierarchy, mixed font sizes, and inconsistent margins

Accessibility & Heuristic Analysis

Using WebAIM WAVE, I discovered several accessibility flaws:

  • Missing or improper alt text on images
  • Low color contrast on key buttons and navigation text
  • Improper heading structure disrupting screen reader flow
  • Redundant or empty links contributing to confusion

These issues severely impacted efficiency and learnability. Users on mobile or with assistive tools would find the interface frustrating or even unusable.

Visual Style Guide

The style guide ensures consistency and accessibility throughout the redesign. Each section was carefully developed in Figma with reusability in mind:

  • Logos: Custom logo redesign for clarity and visual branding. Retained legacy imagery where helpful.
  • Buttons: Designed both default and hover states for clarity, with large touch targets and accessible color contrast.
  • Color Palette: Dark teal and blue provide contrast and consistency. Gold and light grey are used as secondary highlights.
  • Text Styles: Uses Arial for readability with a clear type scale from headings to captions. Font sizes ensure readability across screen sizes.
  • Icons: Clean vector icons for search and shopping cart increase scanability and usability.
  • Controls: Search bar styled for clear interaction, with placeholder text and dropdown support.

Hi-Fidelity Mockups

The design accommodates desktop, tablet, and mobile screens. Elements resize, reposition, or collapse fluidly across breakpoints:

  • Layout Strategy: Used CSS Grid for the header and Flexbox for product listings. Navigation collapses into a single row on smaller screens.
  • Spacing & Typography: Margins and padding adjust to maintain readable white space. Text resizes slightly between breakpoints.
  • Responsiveness: Image gallery stacks vertically on mobile, with full-width buttons for accessibility.
  • Annotated Details: Font sizes, max-widths, and layout behaviors are documented in the Figma file for each view.

Reflection & Takeaways

This project taught me how to design with accessibility and responsiveness in mind from the beginning. Creating annotated mockups and a reusable style system made implementation smoother and allowed for cleaner, more maintainable code. Testing responsiveness with browser tools and feedback from peers helped ensure the design felt natural across devices.
